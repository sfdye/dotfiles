## Features

- Ubuntu & macOS friendly
- zsh
- iterm
- git
- docker
- python3 developmment with virtualenv/virtualenvwrapper
- ruby dev env with rbenv
- awscli and aws-shell
- vscode
- Brewfile

## Installation

```bash
git clone https://github.com/sfdye/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```
